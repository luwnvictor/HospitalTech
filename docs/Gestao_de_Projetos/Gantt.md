# 📊 Cronograma do Projeto (Gráfico de Gantt)

Este repositório foi criado para organizar e facilitar o acesso ao **cronograma do projeto**, estruturado no formato de **Gráfico de Gantt**.

## 📌 Sobre o Cronograma
O cronograma apresenta o planejamento das atividades do projeto, incluindo:
- Etapas e tarefas
- Datas de início e término
- Duração das atividades
- Dependências entre tarefas
