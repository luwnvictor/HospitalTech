# 📄 Termo de Abertura do Projeto (TAP)

Este repositório foi criado com o objetivo de armazenar e organizar o acesso ao **TAP (Termo de Abertura do Projeto)**.

## 📌 Sobre o Projeto
O TAP contém as informações iniciais do projeto, como:
- Objetivo
- Justificativa
- Escopo
- Stakeholders
- Premissas e restrições
